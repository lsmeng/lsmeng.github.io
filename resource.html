Resources

Open-source code from our group for earthquake source imaging. The packages are developed and maintained by group members; see each repository for instructions, demo data, and the related papers. More on our GitHub: github.com/lsmeng.

Code

🛰️

MUSIC & Beamforming Back-Projection

Teleseismic P-wave back-projection in MATLAB — both beamforming and high-resolution MUSIC. Worked example on the 2018 Mw 7.5 Palu (and 2011 Tohoku) earthquakes.
MUSICBP · master branch
🎯

Slowness-Enhanced Back-Projection (SEBP)

MUSIC back-projection with aftershock-based slowness calibration to remove array-dependent spatial bias. Worked example on the 2021 Mw 7.4 Maduo earthquake.
MUSICBP · SEBP branch
💥

Mach-Wave Analysis for Supershear

Python/ObsPy tools to identify the Rayleigh Mach wave of supershear ruptures via waveform similarity between a mainshock and a collocated empirical Green's function event.
MUSICBP · MachWave branch
🤖

Machine-Learning for Seismology

Course/lab materials and machine-learning utilities used in our deep-learning seismology work (e.g., phase picking and earthquake monitoring).
MachineLearning repository

External reference: routine back-projection images of new large earthquakes are produced by EarthScope/IRIS.
